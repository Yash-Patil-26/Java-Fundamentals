# Java Basic Programs
This repository contains simple Java programs created for practice and learning basic programming concepts.

# Topics Covered
* If-else statements
* Loops (for, while)
* Number programs (Prime, Armstrong, Fibonacci, Factorial)
* GCD and LCM
* String operations (remove duplicates, count characters, etc.)
* Pattern programs
* Basic mathematical problems

# Purpose
The main goal of this repository is to:

* Practice Java syntax
* Improve logical thinking
* Understand basic programming concepts

# Note
These are basic standalone programs written for learning purposes.
They are not part of any larger project or application.
